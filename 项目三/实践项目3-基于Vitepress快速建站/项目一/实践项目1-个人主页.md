# 04-实践项目1

## 个人主页项目

### 项目概述

创建一个使用HTML标签构建的个人简历页面，用于展示个人信息、技能和作品。（Tips：巩固HTML基础，熟悉HTML文档结构和常用标签的使用）

### 项目目标

1. 掌握HTML文档的基本结构和语法
2. 能够使用常用的HTML标签构建页面内容
3. 了解HTML语义化的重要性并初步应用
4. 能够使用列表、表格等元素展示结构化数据
5. 掌握图片、链接等媒体元素的使用方法
6. 初步了解表单的创建和使用

### 实践项目步骤

**步骤一：创建HTML文档结构**

首先，我们需要创建一个完整的HTML文档结构，包括DOCTYPE声明、html、head和body标签。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的个人主页</title>
</head>
<body>
    <!-- 页面内容将在这里添加 -->
</body>
</html>
```

**步骤二：添加页面头部**

在body标签内，添加页面头部信息，包括个人姓名、职业和简短介绍。

```html
<header>
    <h1>张三</h1>
    <p>Web前端开发工程师</p>
    <p>热爱技术，专注于创建优秀的用户体验</p>
</header>
```
