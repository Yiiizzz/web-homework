# 实践项目3：基于 VitePress 的快速建站

## 一、项目简介
本项目是《Web 前端开发》课程实践项目三，  
使用 **VitePress** 搭建一个静态网站，用于展示个人信息与学习内容。

项目目标：
- 学会使用 VitePress 构建静态站点
- 理解 docs 目录结构
- 掌握本地运行与静态部署流程
