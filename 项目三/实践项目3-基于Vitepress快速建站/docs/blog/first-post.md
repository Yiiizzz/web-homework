# 我的第一篇博客

> 这篇文章记录我如何用 VitePress 完成站点结构、构建与部署，并解决路径与链接问题。

## 我做了什么
- 初始化项目与目录结构
- 配置导航与侧边栏
- 修复 dead link
- GitHub Pages 部署成功

## 踩坑记录
- `base` 必须与 Pages 路径一致
- 构建时 dead link 会导致失败

## 总结
这次实践让我理解了静态站点生成器的工作流程：Markdown → 构建 → HTML/CSS/JS → 部署。
